## Функциональность страниц

### Страница 1 (β-функция)

- Формула:  
  \[
  \beta = \sqrt{10\left(\sqrt[3]{x} + x^{y+2}\right)(\arcsin^2 z - |x - y|)}
  \]
- Ввод переменных `x`, `y`, `z`.
- Проверка области определения арксинуса (\(z \in [-1, 1]\)) и неотрицательности подкоренного выражения.
- Динамические подсказки при наведении на поля `Y` и `Z`, которые зависят от текущих значений `X` и других переменных (рекомендуемые диапазоны).

### Страница 2 (кусочная функция)

- Формула:
  \[
  g = \begin{cases}
  e^{f(x) - |b|}, & 0.5 < xb < 10 \\
  \sqrt{|f(x)| + |b|}, & 0.1 < xb < 0.5 \\
  2f(x)^2, & \text{иначе}
  \end{cases}
  \]
- Выбор вида функции \(f(x)\) с помощью радиокнопок: `sh(x)`, `x²`, `e^x`.
- Ввод переменных `x` и `b`.
- Подсказки на полях ввода (статические).

### Страница 3 (табулирование функции с графиком)

- Формула:  
  \[
  y = x^2 + \tan\left(5x + \frac{d}{x}\right)
  \]
- Ввод параметров: `X0` (начало отрезка), `Xk` (конец отрезка), `Dx` (шаг), `d` (параметр).
- Автоматическое обновление графика и таблицы значений при изменении любого параметра.
- Отбрасывание точек с делением на ноль, бесконечными значениями или слишком большими значениями (\(|y| > 10^6\)).
- Защита от зависания: если расчётное количество точек превышает 5000, программа выдаёт предупреждение и не строит график.
- График строится с помощью библиотеки OxyPlot.

## Функциональные требования (реализовано)

-  Навигация по трём страницам приложения

-  TextBox для вывода результатов (IsReadOnly = True)
-  Label для подписей элементов

-  Canvas для отрисовки графика на странице 3
-  ToolTip для всех элементов интерфейса
-  Проверка полей ввода на заполненность и корректность данных

## Инструкция по запуску

1. Клонируйте репозиторий

2. Откройте решение `Практическая_работа_4_Кузнецов_Кузьмин` в Visual Studio 2019/2022

3. Восстановите NuGet-пакеты (при необходимости)

4. Запустите проект:
   - Нажмите **F5** для запуска с отладкой
   - Или **Ctrl + F5** для запуска без отладки

5. Введите значения в поля ввода и нажмите кнопку **«Вычислить»**
